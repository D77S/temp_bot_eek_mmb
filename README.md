На скорую руку написанный бот.
Будет круглосуточно работать, скорее всего на купленном хостинге.
При инициализаци создает первоначальную базу данных:
- для каждой из задач:
  - сходить один раз по ссылке задачи, получить ответ (надеемся, что сервер ответа по задаче будет в порядке на данный раз)
  - ответ сервера подвергнуть парсингу, найти только интересующее
  - ответ записать в БД, вмиесте с таймстампом его получения
Далее переходит в бесконечный цикл:
- уснуть на 5 секунд
- для каждой из задач:
  - проверить, надо ли ее делать. Для этого текущее время сравнить с последним таймстампом задачи в БД, и если разница больше, чем дельта для данной задачи, то:
    - приступить к задаче. Запросить ответ от сервера задачи.
    - ответ подвергнуть парсингу, найти только интересующее
    - сравнить найденное и то, что лежит в БД по данной задаче
    - если НЕ совпадает:
      - выдать аларм в чат-бот для человека
      - обновить в БД ответ сервера
    - обновить в БД таймстамп ответа сервера по задаче
